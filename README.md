# Blackjack - Android Uygulaması

## Proje Hakkında
Blackjack Andorid Uygulaması, klasik blackjack mantığını ve sistemini telefon ekranlarınıza taşıyan temel bir Kotlin yazılım çalışmasıdır.

## Kullanılan Teknolojiler

- Kotlin
- Android Studio


## Projenin Amacı
Bu proje, Android uygulama geliştirme becerilerimi sergilemek ve pratik yapma amacıyla
geliştirilmiştir.
